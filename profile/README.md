# Social-Nav: Benchmarks and ROS-powered Agents for Social Navigation

Welcome to Social-Nav, we provide a collection of repositories focused on social navigation research and tooling (Social Force Model controllers, social navigation benchmarks, VLN agents integrated in ROS eco-system. 

## Quick navigation & pointers
- For LISN project-related code and web front-end, see [tvss_nav](https://github.com/Social-Nav/tvss_nav). 
- For Isaac Sim migration, you can track progress in [Arena5.0](https://github.com/Social-Nav/Arena5.0).
